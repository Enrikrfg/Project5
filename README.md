# Project5
Project 5 for DataScience Bootcamp
Se realiza una aplicación en la que muestra un analisis grafico 
de las marcas de carros más utilizadas en EUA.
Se añadieron 3 diferentes tipos de graficos que utilizan botones interactivos y que despliegan las gráficas con el analisis en los siguientes ordenes: 
1. Un encabezado con texto.
2. Un histograma.
3. Un gráfico de dispersión.
4. Un botón o una casilla de verificación
